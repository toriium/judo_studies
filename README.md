# Fundamentos Judo Pie

**Rei**: Saludo

**Ukemi**: Falls
- Mae-ukemi: Forward breakfall
- Yoko-ukemi: Sideways breakfall
- Ushiro-ukemi: backwards breakfall
- Mae-mawari-ukemi forwards rolling breakfall

**Kuzuzhi**: Desequilibrio  
**Tsukuri**: Preparación  
**Kake**: Proyección
